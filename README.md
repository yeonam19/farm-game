# 힐링 농장 (Healing Farm) 🌾

2D 픽셀아트 스타일의 힐링 농장 시뮬레이션 게임입니다.
작물 재배, 동물 사육, 낚시, 자원 채집 등 다양한 농장 활동을 즐길 수 있습니다.

## 기술 스택

- **언어**: Kotlin 1.9.22
- **프레임워크**: LibGDX 1.12.1
- **빌드**: Gradle (Kotlin DSL)
- **플랫폼**: Desktop (LWJGL3)
- **Java**: 11+

## 실행 방법

### 요구사항

- Java 11 이상 (JDK)

### 게임 실행

```bash
./gradlew desktop:run
```

### 배포용 JAR 빌드

```bash
./gradlew desktop:dist
```

빌드 결과물: `desktop/build/libs/farm-game-1.0.0.jar`

```bash
java -jar desktop/build/libs/farm-game-1.0.0.jar
```

> macOS에서는 `-XstartOnFirstThread` 플래그가 자동 적용됩니다.

## 게임 조작법

| 키 | 동작 |
|---|------|
| 방향키 / WASD | 캐릭터 이동 |
| 숫자키 (1-9) | 도구/씨앗 선택 |
| Space | 상호작용 (심기, 수확, 낚시, 구매 등) |

## 게임 특징

### 농장 시스템
- **작물 재배**: 밭 갈기 → 씨앗 심기 → 물주기 → 수확
- **8종 작물**: 딸기, 감자, 토마토, 옥수수, 당근, 배추, 무, 시금치
- **계절별 재배**: 봄/여름/가을/겨울 각 28일, 계절에 맞는 작물만 성장

### 동물 사육
- **4종 동물**: 닭(달걀), 소(우유), 양(양털), 돼지(송로버섯)
- 구매 후 먹이 주기로 생산품 수확

### 낚시
- 물가에서 낚싯줄 던지기 → 입질 대기 → 낚아올리기
- 5종 물고기 + 쓰레기 아이템

### 자원 채집
- 나무 벌목 (목재 획득)
- 바위 채굴 (돌 획득)

### 경제
- 시장에서 작물/아이템 판매
- 도구 업그레이드, 농장 확장, 동물 구매

### 시간 & 날씨
- 하루 = 실시간 12분
- 날씨: 맑음, 흐림, 비, 폭풍, 눈
- 비 오는 날은 자동 물주기

### 성장 요소
- **레벨 시스템**: 농사/낚시/채집으로 경험치 획득
- **미션 체인**: 20단계 연속 미션
- **업적**: 7개 카테고리, 29개 업적
- **업그레이드**: 도구 강화, 농장 확장, 스프링클러, 자동낚시기
- **고용**: 농부/목동 고용으로 작업 자동화

## 프로젝트 구조

```
farm-game/
├── core/                        # 게임 로직 모듈
│   └── src/main/kotlin/com/farmgame/
│       ├── FarmGame.kt          # 메인 게임 클래스
│       ├── data/                # 게임 데이터 정의
│       ├── entity/              # 엔티티 (Player, Crop, Animal, NPC)
│       ├── screen/              # 화면 렌더링
│       ├── system/              # 게임 시스템 (시간, 날씨, 경제, 미션, 업적, 저장)
│       ├── ui/                  # UI (HUD, 폰트)
│       └── world/               # 월드 (타일, 맵)
├── desktop/                     # 데스크톱 런처 모듈
├── assets/                      # 게임 리소스
│   ├── korean.ttf               # 한글 폰트
│   └── saves/                   # 세이브 파일 (7슬롯)
├── build.gradle.kts             # 루트 빌드 설정
└── settings.gradle.kts          # 모듈 정의
```
